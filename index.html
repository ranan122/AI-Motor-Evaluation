Can AI code infant motor behavior from video?

A structured evaluation framework for measuring how well large language models perform on behavioral coding tasks traditionally done by human researchers, benchmarked on infant sitting behavior.

View on GitHub Read the framework

Evaluation Framework

Two tiers of tasks

The framework separates AI evaluation into two tiers. Tier 1 tasks ask the model to replicate what a trained human coder does: identifying postures, estimating durations, and categorizing infants. Tier 2 tasks ask the model to go further, drawing inferences from visual cues that coders themselves might not explicitly record.

This separation is intentional. Tier 1 has clear ground truth from existing coded data, making accuracy straightforward to measure. Tier 2 requires new scoring rubrics and is inherently harder, but opens the door to questions about what AI can do that traditional coding pipelines cannot.

Tier 1

Coding tasks

Replicating the work of human coders, scoreable against existing ground truth.

  • Posture identification — classify each video frame into postures 0 through 3
  • Temporal estimation — estimate how long each posture lasts within a clip
  • Sitter categorization — classify the infant as an independent, emergent, or non-sitter
Tier 2

Inference tasks

Reasoning beyond explicit coding, requiring new rubrics and scoring approaches.

  • Age estimation — estimate the infant's age in months from motor and physical cues
  • Sex identification — identify the infant's sex from available contextual cues

Sitter Classification

How infants are classified

Infants are classified into one of three sitting categories based on the proportion of codable postural time. Only postures 0 through 3 count toward the denominator — transitional codes F (fall) and N (non-codable) are excluded. Categories are mutually exclusive and checked in order.

Category Rule
Independent sitter Posture 3 accounts for 50% or more of codable time
Emergent sitter Postures 2 and 3 combined account for 50% or more of codable time, but posture 3 alone is under 50%
Non-sitter Postures 0 and 1 combined account for 50% or more of codable time
Posture codes: 0 = lying or unsupported flat  ·  1 = semi-supported  ·  2 = supported sitting  ·  3 = independent sitting  ·  F = transitional fall  ·  N = non-codable transition

Repository

AI performance on infant motor behavior coding.
